Riwayat Pendidikan

  • 2008 - 2009

    Saya memulai Pendidikan Ketika berusia 5 tahun di TK Anakita yang berlokasi di Sengkang,Kab. Wajo. Saya menempuh Pendidikan TK selama setahun. Sejak TK, saya sudah menyukai kesenian, sampai akhirnya saya ditunjuk untuk menari di acara perpisahan kami

  • 2009 - 2014

    Setelah lulus dari Pendidikan di TK Anakita, saya melanjutkan Pendidikan ke tingkat Sekolah Dasar pada tahun 2009, tepatnya di SDN 2 Maddukkeleng. Saya menempuh Pendidikan SD selama 5 tahun. Hal itu karena saya mengikuti program kelas Akselerasi, sehingga mengalami percepatan setahun. Sejak SD, saya mulai mengikuti kegiatan kesenian da juga lomba akademik. Seperti lomba bernyanyi dan juga lomba mata pelajaran. Selain itu, pada saat perpisahan SD, saya ditunjuk untuk mengisi acara hiburan yaitu menyanyi da juga tari kreasi. Selama SD, saya selalu menduduki peringkat 3 besar.

  • 2014 - 2016

    Setelah lulus Sekolah Dasar, saya melanjutkan Pendidikan saya di SMPN 1 Sengkang. Saya menempuh Pendidikan SMP selama 2 tahun. Lagi lagi, saya mengikuti program kelas Akselerasi juga, sehingga saya tidak lagi menempuh Pendidikan SMP selama 3 tahun melainkan hanya 2 tahun. Di SMP, saya juga mengikuti lomba nyanyi untuk mewakili teman kelompok saya pada saat awal masuk SMP tepatnya pada kegiatan Masa Orientasi dan Pengenalan Lingkungan Sekolah. Di SMP, saya juga selalu menduduki peringkat 5 besar, dan pada saat pengumuman kelulusan, saya berhasil meraih peringkat kedua nilai terbaik untuk Ujian Sekolah. Saya juga aktif mengikuti kegiatan organisasi, seperti OSIS.

  • 2016 - 2019

    Setelah lulus Sekolah Menengah Pertama, saya melanjutkan Pendidikan saya di SMAN 7 Wajo. Saya menempuh Pendidikan SMA selama 3 tahun, yang tidak lagi mengikuti program kelas Akselerasi, melainkan kelas khusus. Tesnya sama saja untuk memasuki program kelas Akselerasi, namun bedanya sudah tidak mengalami percepatan. Di SMA, saya aktif juga berorganisasi seperti Karya Ilmiah Remaja (KIR) dan Pusat Indormasi da Konseling Remaja (PIK-R). Saya juga mengikuti event nasional, yaitu Positive Youth Project.

  • 2019 - Sekarang

    Saat ini, saya tengah menempuh Pendidikan di Universitas Hasanuddin, Fakultas Pertanian, Program Studi Agribisnis.
